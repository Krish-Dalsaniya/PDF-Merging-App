# PDF Merging App

A simple web app to merge multiple PDF files into a single PDF quickly and easily.

## 🚀 Features

- Upload multiple PDF files
- Reorder files before merging (if supported)
- Merge PDFs into one combined file
- Download the merged PDF
- Clean and lightweight interface

## 🛠️ Tech Stack

- **JavaScript** (main logic)
- **HTML**
- **CSS**

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Krish-Dalsaniya/PDF-Merging-App.git
cd PDF-Merging-App
```

### 2. Run the project

If it is a plain frontend project, simply open the main HTML file in your browser:

```bash
open index.html
```

Or use VS Code Live Server for better development experience.

## 📦 Usage

1. Open the app in your browser.
2. Upload the PDF files you want to combine.
3. Arrange files in the desired order.
4. Click the **Merge** button.
5. Download the merged PDF.
